# Agent Hooks & Callbacks

Hooks allow you to intercept and modify agent behavior at key points.

---

## Core Hooks

### `convertToLlm(messages)`

**Purpose:** Transform your app's message format to LLM format.

**When called:** Every time the agent makes an LLM request.

**Signature:**

```typescript
(messages) => messages
```

**Example:**

```typescript
convertToLlm: (messages) => {
    return {
        messages: [
            { role: 'system', content: 'You are a helpful assistant' },
            ...messages
        ]
    }
}
```

---

### `transformContext(messages, signal)`

**Purpose:** Prune/compact conversation history before LLM call.

**When called:** Before every LLM request (to control token usage).

**Signature:**

```typescript
(messages, signal) => messages
```

**Example:**

```typescript
transformContext: (messages, signal) => {
    if (messages.length > 20) {
        return messages.slice(-20);
    }
    return messages;
}
```

---

## Tool Execution Hooks

### `beforeToolCall(definition)`

**Purpose:** Block dangerous tool execution.

**When called:** Before any tool execution starts.

**Signature:**

```typescript
(definition) => boolean
```

**Example:**

```typescript
beforeToolCall: (definition) => {
    if (definition.name === 'bash') {
        return false;  // Block dangerous shell execution
    }
    return true;
}
```

---

### `afterToolCall(result)`

**Purpose:** Add metadata/logging to tool results.

**When called:** After tool execution completes.

**Signature:**

```typescript
(result) => object
```

**Example:**

```typescript
afterToolCall: (result) => {
    result.metadata = { timestamp: new Date() };
    return result;
}
```

---

## Steering Hooks

### `onAgentEnd()`

**Purpose:** Called when agent execution would terminate.

**When called:** Between loops after `agentLoop()` completes.

### `onSteeringEnd()`

**Purpose:** Called when agent run ends (normal or via abort).

**When called:** At the end of the entire agent run.

---

## Message Update Hooks

### `onMessageUpdate(message)`

**Purpose:** Stream partial LLM responses to user.

**When called:** During LLM streaming.

**Example:**

```typescript
onMessageUpdate: (message) => {
    // Append partial response to UI
    ui.append(message.content);
}
```
